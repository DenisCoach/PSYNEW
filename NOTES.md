# PSYNEW — Time Tracker Bot

## Что это
Телеграм бот для отслеживания времени. Каждый час спрашивает чем занимался, накапливает контексты, показывает статистику.

## Бот
- Username: @Psynewtestbot
- Репозиторий: https://github.com/DenisCoach/PSYNEW

## Как запустить локально
```bash
cd "/Users/mac/Documents/Programming/GidHub/my Stars project/my project/New Psy"
python3 main.py
```

## Как работает
- Уведомления каждый час с 11:00 до 21:00 (спрашивает о предыдущем часе)
- Пользователь вводит: описание → время → контекст
- Контексты накапливаются и предлагаются на выбор
- Каждый контекст получает свой цвет (эмодзи)
- Мульти-пользовательский, у каждого свой часовой пояс

## Команды бота
| Команда | Действие |
|---------|----------|
| /start | Регистрация + выбор часового пояса |
| /add | Добавить дело вручную (текущий час) |
| /stats | Статистика: день / неделя / месяц |
| /timezone | Сменить часовой пояс |
| /cancel | Отменить текущее действие |

## Структура файлов
```
main.py        — точка входа
config.py      — настройки, часовые пояса, цвета контекстов
database.py    — вся работа с SQLite (bot.db)
handlers.py    — команды, FSM-диалоги, форматирование статистики
keyboards.py   — инлайн-клавиатуры
scheduler.py   — отправка уведомлений каждую минуту (проверка часового пояса)
states.py      — состояния FSM (Registration, ActivityFSM)
requirements.txt
.env           — токен бота (не в git!)
.env.example   — шаблон
```

## Стек
- Python 3.9+
- aiogram 3.13 — Telegram Bot framework
- aiosqlite — асинхронная работа с SQLite
- APScheduler 3.10 — планировщик уведомлений
- pytz — работа с часовыми поясами
